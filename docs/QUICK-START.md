## 🗺️ เมนูหลัก (แถบบน)

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo] Dashboard │ ใบ PO │ รอรับของ │ เบิกของ │ Catalog* │ ... │ 🔔 │ 👤 │
└─────────────────────────────────────────────────────────────────┘
                                          (* = privileged เท่านั้น)
```

| เมนู | ใครเห็น | ใช้ทำอะไร |
|---|---|---|
| **Dashboard** | ทุกคน | ภาพรวมของฉัน + งานที่รอ |
| **ใบ PO** | ทุกคน | ดู/สร้างใบสั่งซื้อ |
| **รอรับของ** | ทุกคน | กดรับของจาก supplier |
| **เบิกของ** | ทุกคน | เบิกของจาก stock |
| **Catalog** | Privileged | จัดการสินค้าในระบบ |
| **งบ** | Privileged | ตั้งงบประมาณ |
| **รายงาน** | Privileged | สถิติการสั่งซื้อ |
| **ผู้ใช้** | Privileged | จัดการ user |

## 🎯 3 Tasks ที่ใช้บ่อยสุด

### ➕ สร้างใบ PO ใหม่
1. กดเมนู **ใบ PO** → ปุ่ม **+ สร้างใบ PO ใหม่**
2. เลือก equipment จาก catalog (กดเพิ่มจำนวนที่ต้องการ)
   หรือกด **"พิมพ์ชื่อเอง"** → กรอก + แนบรูป (ระบบบีบรูปอัตโนมัติ)
3. ดู cart ด้านบน → กด **"บันทึกใบ PO"**
4. ✅ เสร็จ → admin จะได้ notification ไปสั่งซื้อ

### 📦 รับของ (เมื่อ supplier ส่งมา)
1. กดเมนู **รอรับของ** → คลิก PO ที่จะรับ
   - ปุ่ม **"รับของ"** จะปรากฏเฉพาะเมื่อสถานะ "กำลังขนส่ง"
2. กรอกจำนวนรับจริง + จำนวนที่เสียหาย
3. แนบรูปประกอบ (ถ้ามี — ระบบบีบให้)
4. กด **"บันทึก"**
5. ✅ Stock จะเพิ่มอัตโนมัติ

### 🔧 เบิกของจาก Stock
1. กดเมนู **เบิกของ**
2. เลือกสินค้าจาก catalog → ใส่จำนวน
3. กรอกเหตุผล (เช่น "เบิกใช้ในห้อง lab")
4. กด **"บันทึก"**
5. ✅ Stock ลดอัตโนมัติ

## 🆘 ปัญหาที่เจอบ่อย

| ปัญหา | แก้ |
|---|---|
| Login ไม่ได้ — ลืมรหัส | ติดต่อ Admin ให้ reset ให้ |
| ไฟล์รูปใหญ่ upload ไม่ได้ | ระบบบีบรูปอัตโนมัติ — ลอง refresh แล้วลองใหม่ |
| กดรับของไม่ได้ — ไม่มีปุ่ม | รอ admin อัปเดตสถานะเป็น "กำลังขนส่ง" ก่อน |
| ไม่ได้รับอีเมล | ลอง spam folder + ติดต่อ admin ตรวจ SMTP |
| หน้าโหลดช้า | กด Cmd/Ctrl+Shift+R เพื่อ refresh แบบ hard |
