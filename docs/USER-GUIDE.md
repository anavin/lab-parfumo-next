│                                         │

│                                         │
│   [        เข้าสู่ระบบ        ]          │
│                                         │
│   ℹ️ บัญชีเริ่มต้น (ถ้า admin ตั้งไว้)    │
│   admin / admin123 → Admin              │

**ขั้นตอน:**
1. กรอก **Username** + **Password** (admin จะแจ้งให้)
2. กด **เข้าสู่ระบบ**
3. ครั้งแรก → ระบบจะ redirect ไปหน้าเปลี่ยน password ทันที

```

│                                         │
│   รหัสผ่านปัจจุบัน:                      │

│   [____________________________]        │
│   ⓘ อย่างน้อย 8 ตัว มีตัวอักษร + ตัวเลข   │
│                                         │
│   ยืนยันรหัสผ่านใหม่:                    │

│                                         │

└─────────────────────────────────────────┘
```

**กฎการตั้ง password:**
- ✅ อย่างน้อย **8 ตัวอักษร**

- ❌ ห้ามเหมือน username
- ❌ ห้ามใช้รหัสยอดนิยม (`password`, `12345678`, `admin123`)

**ขั้นตอน:**
1. เลื่อนดู catalog หรือใช้ search bar หาสินค้า
2. คลิกหมวด (📦 บรรจุภัณฑ์, 🧪 สารเคมี ฯลฯ) → กดย่อขยาย

- ❌ ลบสินค้าออก cart: กด `[-]` ลดจำนวนเป็น 0

- ⌨️ Esc เพื่อปิด lightbox

## 6. สร้างใบ PO Custom Item

หากสินค้าที่ต้องการ**ไม่มีใน catalog** — สามารถพิมพ์ชื่อเองได้:

```
┌────────────────────────────────────────────────────────┐
│ ▼ ✏️ พิมพ์ชื่อเอง (สำหรับรายการที่ไม่มีใน catalog)        │
│                                                        │
│ ชื่อรายการ * [_____________________________]           │
│ จำนวน * [10] หน่วย [ชิ้น]                              │
│                                                        │
│ 📷 รูปประกอบ (แนะนำ — ให้ admin เห็นว่าจะสั่งอะไร)        │
│ ┌──────────────────────────────────────────────────┐   │
│ │ 🔄 กำลังเตรียมรูป... 1 / 2                        │   │
│ │ ━━━━━━━━━━ 50%                                    │   │
│ │ 💡 บีบรูปขนาดใหญ่ใน browser — ภาพยังชัดเหมือนเดิม  │   │
│ └──────────────────────────────────────────────────┘   │
│                                                        │
│ ✅ ประหยัด 8 → 0.7 MB (91%) • 2 รูป                     │
│                                                        │
│ [ + เพิ่มเข้าใบ PO ]                                     │
└────────────────────────────────────────────────────────┘
```

**ขั้นตอน:**
1. กดเปิด section **"พิมพ์ชื่อเอง"**
2. กรอกชื่อสินค้า + จำนวน + หน่วย + หมายเหตุ
3. คลิกเลือกรูปจากเครื่อง (รองรับ HEIC/PNG/JPG)
4. ระบบ**บีบรูปอัตโนมัติ** ใน browser (~1-2 วินาที/รูป)
5. ดู savings summary
6. กด **"+ เพิ่มเข้าใบ PO"**

**Tips:**
- 📷 iPhone HEIC → ระบบแปลงเป็น JPEG ให้
- 🖼️ ขนาดสูงสุด 1920px (พอแสดงทุกหน้าจอ)
- ⚠️ ระบบบีบรูป — ใหญ่ถึง 50 MB ก็ได้, จะลดเหลือ ~500 KB

# 🆘 FAQ

## 26. ปัญหาที่พบบ่อย

### Q1: Login ไม่ได้ — ลืมรหัสผ่าน
A: ติดต่อ admin → admin จะ reset password ให้คุณผ่าน /users → คุณจะได้ temp password ใหม่ → login → ระบบบังคับเปลี่ยน

### Q2: Login ไม่ได้ — บัญชีถูกล็อค
A: หลัง login ผิด 5 ครั้งต่อเนื่องใน 15 นาที → บัญชีจะถูกล็อค **15 นาที** อัตโนมัติ → รอแล้วลองใหม่ หรือติดต่อ admin

### Q3: ส่งรูปไม่สำเร็จ — ไฟล์ใหญ่
A: ระบบ**บีบรูปอัตโนมัติ** — ใหญ่เท่าไหร่ก็ได้ ลอง refresh แล้วเลือกใหม่ ถ้ายังไม่ได้ → ตรวจ network

### Q4: HEIC จาก iPhone ส่งไม่ได้?
A: ระบบ**แปลง HEIC → JPEG อัตโนมัติ** ใน browser. ถ้ายังไม่ได้ → browser อาจเก่า — ลอง Safari/Chrome ล่าสุด

### Q5: กดรับของไม่ได้ — ไม่มีปุ่ม
A: ปุ่มรับของจะปรากฏ**เฉพาะเมื่อ status = "กำลังขนส่ง"** — ถ้าเป็น "สั่งซื้อแล้ว" ต้องรอ admin/supervisor กด **อัปเดตขนส่ง** ก่อน

### Q6: ไม่ได้รับอีเมล welcome
A: 
1. เช็ค **Spam folder** (เข้าบ่อยที่สุด)
2. ติดต่อ admin → ตรวจ /settings → อีเมล (SMTP) → กดทดสอบ
3. ตรวจ email ที่กรอกถูกต้อง

### Q7: PO ของฉันหายไป
A: 
1. เช็ค filter ตอน list (อาจเลือก "เสร็จสมบูรณ์" อยู่)
2. ถ้าเป็น Staff — เห็นแค่ของตัวเอง (admin จะเห็นทั้งหมด)
3. ถ้ายังไม่เจอ — ติดต่อ admin

### Q8: หน้าโหลดช้า
A:
1. กด **Cmd/Ctrl+Shift+R** (hard refresh)
2. เช็ค network connection
3. ถ้ายังช้า — แจ้ง admin

### Q9: ข้อมูลใน Dashboard ไม่อัปเดต
A: F5 refresh หน้า — ระบบใช้ cache สั้นๆ เพื่อให้เร็ว แต่จะ refresh ทันทีถ้ามี action

### Q10: Stock ผิด — ตัวเลขไม่ตรง
A: 
1. เช็คประวัติเบิก + รับของใน activity log
2. ถ้ายกเลิก PO ที่รับของแล้ว — ระบบควรถอย stock อัตโนมัติ
3. ถ้าผิด — ติดต่อ admin → admin แก้ stock ตรงๆ ใน /equipment
